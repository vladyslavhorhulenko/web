МІЙ ЕЛЕКТРОННИЙ ДОВІДНИК HTML
Призначення Назва тегу Короткий опис
Для каркасу документа <html> Позначає початок та кінець всієї html-сторінки
<head> Розділ сторінки для метаданих та підключення сторонніх файлів
<body> Тіло документу
Для роботи з текстом <p> Позначає відкриття та закриття абзацу
<b> Робить текст жирним
<i> Переводить текст в курсив
<u> Підкреслює текст
Для гіперпосилань <a> Позначає початок та кінець гіперпосилання
